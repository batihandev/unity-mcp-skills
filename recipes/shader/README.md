# shader recipes

Recipe path rule: `../../recipes/shader/<command>.md`

| Command | File | Description |
|---------|------|-------------|
| `shader_create` | [shader_create.md](shader_create.md) | Create a shader file from template |
| `shader_read` | [shader_read.md](shader_read.md) | Read shader source code |
| `shader_list` | [shader_list.md](shader_list.md) | List all shaders in project |
| `shader_find` | [shader_find.md](shader_find.md) | Find a shader by name |
| `shader_delete` | [shader_delete.md](shader_delete.md) | Delete a shader file |
| `shader_get_properties` | [shader_get_properties.md](shader_get_properties.md) | Get all properties defined in a shader |
| `shader_check_errors` | [shader_check_errors.md](shader_check_errors.md) | Check shader for compilation errors |
| `shader_get_keywords` | [shader_get_keywords.md](shader_get_keywords.md) | Get shader keyword list |
| `shader_get_variant_count` | [shader_get_variant_count.md](shader_get_variant_count.md) | Get shader variant count for performance analysis |
| `shader_create_urp` | [shader_create_urp.md](shader_create_urp.md) | Create a URP shader from template (Unlit or Lit) |
| `shader_set_global_keyword` | [shader_set_global_keyword.md](shader_set_global_keyword.md) | Enable or disable a global shader keyword |
