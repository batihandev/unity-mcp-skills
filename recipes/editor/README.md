# Editor Recipes

Per-command recipes for the `unity-editor` skill. Each file corresponds to one command ID.

## Commands

| File | Command |
|------|---------|
| [editor_play.md](editor_play.md) | `editor_play` |
| [editor_stop.md](editor_stop.md) | `editor_stop` |
| [editor_pause.md](editor_pause.md) | `editor_pause` |
| [editor_select.md](editor_select.md) | `editor_select` |
| [editor_get_selection.md](editor_get_selection.md) | `editor_get_selection` |
| [editor_get_context.md](editor_get_context.md) | `editor_get_context` |
| [editor_undo.md](editor_undo.md) | `editor_undo` |
| [editor_redo.md](editor_redo.md) | `editor_redo` |
| [editor_get_state.md](editor_get_state.md) | `editor_get_state` |
| [editor_execute_menu.md](editor_execute_menu.md) | `editor_execute_menu` |
| [editor_get_tags.md](editor_get_tags.md) | `editor_get_tags` |
| [editor_get_layers.md](editor_get_layers.md) | `editor_get_layers` |

## Usage

Use these templates in `Unity_RunCommand`. Recipe path rule: `../../recipes/editor/<command>.md`
