# terrain recipes

Recipe path rule: `../../recipes/terrain/<command>.md`

| Command | File | Description |
|---------|------|-------------|
| `terrain_create` | [terrain_create.md](terrain_create.md) | Create a new Terrain GameObject with a TerrainData asset |
| `terrain_get_info` | [terrain_get_info.md](terrain_get_info.md) | Get terrain size, resolution, and layer information |
| `terrain_get_height` | [terrain_get_height.md](terrain_get_height.md) | Sample terrain height at a world position |
| `terrain_set_height` | [terrain_set_height.md](terrain_set_height.md) | Set terrain height at a world position |
| `terrain_set_heights_batch` | [terrain_set_heights_batch.md](terrain_set_heights_batch.md) | Set multiple terrain heights in a single call |
| `terrain_add_hill` | [terrain_add_hill.md](terrain_add_hill.md) | Add a Gaussian hill to the terrain at a world position |
| `terrain_generate_perlin` | [terrain_generate_perlin.md](terrain_generate_perlin.md) | Generate terrain heightmap using Perlin noise |
| `terrain_smooth` | [terrain_smooth.md](terrain_smooth.md) | Smooth a region of the terrain heightmap |
| `terrain_flatten` | [terrain_flatten.md](terrain_flatten.md) | Flatten the entire terrain to a uniform height |
| `terrain_paint_texture` | [terrain_paint_texture.md](terrain_paint_texture.md) | Paint a texture layer onto the terrain |
