# NavMesh Recipes

Per-command recipes for the `unity-navmesh` skill. Each file corresponds to one command ID.

## Commands

| File | Command |
|------|---------|
| [navmesh_bake.md](navmesh_bake.md) | `navmesh_bake` |
| [navmesh_clear.md](navmesh_clear.md) | `navmesh_clear` |
| [navmesh_calculate_path.md](navmesh_calculate_path.md) | `navmesh_calculate_path` |
| [navmesh_add_agent.md](navmesh_add_agent.md) | `navmesh_add_agent` |
| [navmesh_set_agent.md](navmesh_set_agent.md) | `navmesh_set_agent` |
| [navmesh_add_obstacle.md](navmesh_add_obstacle.md) | `navmesh_add_obstacle` |
| [navmesh_set_obstacle.md](navmesh_set_obstacle.md) | `navmesh_set_obstacle` |
| [navmesh_sample_position.md](navmesh_sample_position.md) | `navmesh_sample_position` |
| [navmesh_set_area_cost.md](navmesh_set_area_cost.md) | `navmesh_set_area_cost` |
| [navmesh_get_settings.md](navmesh_get_settings.md) | `navmesh_get_settings` |

## Usage

Use these templates in `Unity_RunCommand`. Recipe path rule: `../../recipes/navmesh/<command>.md`
