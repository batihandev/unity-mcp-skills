# Light Recipes

Per-command recipes for the `unity-light` skill. Each file corresponds to one command ID.

## Commands

| File | Command |
|------|---------|
| [light_create.md](light_create.md) | `light_create` |
| [light_set_properties.md](light_set_properties.md) | `light_set_properties` |
| [light_set_properties_batch.md](light_set_properties_batch.md) | `light_set_properties_batch` |
| [light_set_enabled.md](light_set_enabled.md) | `light_set_enabled` |
| [light_set_enabled_batch.md](light_set_enabled_batch.md) | `light_set_enabled_batch` |
| [light_get_info.md](light_get_info.md) | `light_get_info` |
| [light_find_all.md](light_find_all.md) | `light_find_all` |
| [light_add_probe_group.md](light_add_probe_group.md) | `light_add_probe_group` |
| [light_add_reflection_probe.md](light_add_reflection_probe.md) | `light_add_reflection_probe` |
| [light_get_lightmap_settings.md](light_get_lightmap_settings.md) | `light_get_lightmap_settings` |

## Usage

Use these templates in `Unity_RunCommand`. Recipe path rule: `../../recipes/light/<command>.md`
