# sample recipes

Recipe path rule: `../../recipes/sample/<command>.md`

| Command | File | Description |
|---------|------|-------------|
| `create_cube` | [create_cube.md](create_cube.md) | Create a cube primitive at the specified position |
| `create_sphere` | [create_sphere.md](create_sphere.md) | Create a sphere primitive at the specified position |
| `delete_object` | [delete_object.md](delete_object.md) | Delete a GameObject by name |
| `find_objects_by_name` | [find_objects_by_name.md](find_objects_by_name.md) | Find all GameObjects containing a name string |
| `get_scene_info` | [get_scene_info.md](get_scene_info.md) | Get current scene information |
| `set_object_position` | [set_object_position.md](set_object_position.md) | Set world-space position of a GameObject |
| `set_object_rotation` | [set_object_rotation.md](set_object_rotation.md) | Set rotation of a GameObject (Euler angles) |
| `set_object_scale` | [set_object_scale.md](set_object_scale.md) | Set local scale of a GameObject |
