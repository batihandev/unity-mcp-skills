# Event Recipes

Per-command recipes for the `unity-event` skill. Each file corresponds to one command ID.

| File | Command |
|------|---------|
| [event_get_listeners.md](event_get_listeners.md) | `event_get_listeners` |
| [event_add_listener.md](event_add_listener.md) | `event_add_listener` |
| [event_remove_listener.md](event_remove_listener.md) | `event_remove_listener` |
| [event_invoke.md](event_invoke.md) | `event_invoke` |
| [event_clear_listeners.md](event_clear_listeners.md) | `event_clear_listeners` |
| [event_set_listener_state.md](event_set_listener_state.md) | `event_set_listener_state` |
| [event_list_events.md](event_list_events.md) | `event_list_events` |
| [event_add_listener_batch.md](event_add_listener_batch.md) | `event_add_listener_batch` |
| [event_copy_listeners.md](event_copy_listeners.md) | `event_copy_listeners` |
| [event_get_listener_count.md](event_get_listener_count.md) | `event_get_listener_count` |

## Usage

Use these templates in `Unity_RunCommand`. Recipe path rule: `../../recipes/event/<command>.md`
