# Scene Recipes

Per-command recipes for the `unity-scene` skill. Each file corresponds to one command ID.

## Commands

| File | Command |
|------|---------|
| [scene_create.md](scene_create.md) | `scene_create` |
| [scene_load.md](scene_load.md) | `scene_load` |
| [scene_save.md](scene_save.md) | `scene_save` |
| [scene_get_info.md](scene_get_info.md) | `scene_get_info` |
| [scene_get_hierarchy.md](scene_get_hierarchy.md) | `scene_get_hierarchy` |
| [scene_screenshot.md](scene_screenshot.md) | `scene_screenshot` |
| [scene_get_loaded.md](scene_get_loaded.md) | `scene_get_loaded` |
| [scene_unload.md](scene_unload.md) | `scene_unload` |
| [scene_set_active.md](scene_set_active.md) | `scene_set_active` |
| [scene_find_objects.md](scene_find_objects.md) | `scene_find_objects` |

## Usage

Use these templates in `Unity_RunCommand`. Recipe path rule: `../../recipes/scene/<command>.md`
