# material recipes

Recipe path rule: `../../recipes/material/<command>.md`

| Command | File | Description |
|---------|------|-------------|
| `material_create` | [material_create.md](material_create.md) | Create a new material (auto-detects render pipeline) |
| `material_create_batch` | [material_create_batch.md](material_create_batch.md) | Create multiple materials in one call |
| `material_assign` | [material_assign.md](material_assign.md) | Assign a material asset to a renderer |
| `material_assign_batch` | [material_assign_batch.md](material_assign_batch.md) | Assign materials to multiple objects in one call |
| `material_duplicate` | [material_duplicate.md](material_duplicate.md) | Duplicate an existing material asset |
| `material_set_color` | [material_set_color.md](material_set_color.md) | Set a color property with optional HDR intensity |
| `material_set_colors_batch` | [material_set_colors_batch.md](material_set_colors_batch.md) | Set colors on multiple objects in one call |
| `material_set_emission` | [material_set_emission.md](material_set_emission.md) | Set emission color with HDR intensity and auto-enable keyword |
| `material_set_emission_batch` | [material_set_emission_batch.md](material_set_emission_batch.md) | Set emission on multiple objects in one call |
| `material_set_texture` | [material_set_texture.md](material_set_texture.md) | Set a texture on a material |
| `material_set_texture_offset` | [material_set_texture_offset.md](material_set_texture_offset.md) | Set texture UV offset |
| `material_set_texture_scale` | [material_set_texture_scale.md](material_set_texture_scale.md) | Set texture UV scale (tiling) |
| `material_set_float` | [material_set_float.md](material_set_float.md) | Set a float property (e.g. `_Metallic`, `_Smoothness`) |
| `material_set_int` | [material_set_int.md](material_set_int.md) | Set an integer property |
| `material_set_vector` | [material_set_vector.md](material_set_vector.md) | Set a Vector4 property |
| `material_set_keyword` | [material_set_keyword.md](material_set_keyword.md) | Enable or disable a shader keyword |
| `material_set_render_queue` | [material_set_render_queue.md](material_set_render_queue.md) | Set material render queue |
| `material_set_shader` | [material_set_shader.md](material_set_shader.md) | Change the shader of a material |
| `material_set_gi_flags` | [material_set_gi_flags.md](material_set_gi_flags.md) | Set global illumination flags |
| `material_get_properties` | [material_get_properties.md](material_get_properties.md) | Get all material properties |
| `material_get_keywords` | [material_get_keywords.md](material_get_keywords.md) | Get enabled shader keywords |
