# smart recipes

Recipe path rule: `../../recipes/smart/<command>.md`

| Command | File | Description |
|---------|------|-------------|

| `smart_scene_layout` | [smart_scene_layout.md](smart_scene_layout.md) | Arrange selected objects in a layout (Linear, Grid, Circle, Arc) |
| `smart_reference_bind` | [smart_reference_bind.md](smart_reference_bind.md) | Auto-fill a List/Array field with matching scene objects |
| `smart_scene_query_spatial` | [smart_scene_query_spatial.md](smart_scene_query_spatial.md) | Find objects within a sphere region |
| `smart_align_to_ground` | [smart_align_to_ground.md](smart_align_to_ground.md) | Raycast selected objects downward to align to ground |
| `smart_distribute` | [smart_distribute.md](smart_distribute.md) | Evenly distribute selected objects between first and last positions |
| `smart_snap_to_grid` | [smart_snap_to_grid.md](smart_snap_to_grid.md) | Snap selected objects to a grid |
| `smart_randomize_transform` | [smart_randomize_transform.md](smart_randomize_transform.md) | Randomize position/rotation/scale of selected objects |
| `smart_replace_objects` | [smart_replace_objects.md](smart_replace_objects.md) | Replace selected objects with a prefab |
| `smart_select_by_component` | [smart_select_by_component.md](smart_select_by_component.md) | Select all objects that have a specific component |
